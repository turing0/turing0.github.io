    双重积分\iint：

    行内积分limits模式\int\limits_{}^{}：$\int\limits_{-\infty}^{+\infty} f(x) \mathrm{d}x$
    行内积分display模式\displaystyle \int_{}^{}：$\displaystyle \int_{-\infty}^{+\infty} f(x) \mathrm{d}x$

    行内求和display模式\displaystyle \sum_{}^{}：$\displaystyle \sum_{i=1}^{n} i^2$
